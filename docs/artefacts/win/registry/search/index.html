ACMRU

“Search History” on the Windows system (via Search Assistant)

🔑 NTUSER.DAT\Software\Microsoft\SearchAssistant\ACMru\XXXX. Possible values of XXXX:

  1. Internet search - 5001
  2. All or part of the doc name - 5603
  3. A word or phrase in a file - 5604
  4. Printers, computers and people - 5647
