radare2 🍏, 🐧, 🌈 , 📱

Absoultely free that’s why it’s such a darling for many people. However its a command line tool with lots of commands to keep in mind which might make the analysis slower if you are not using it pretty oftern. Also, I am not sure whether they have some scripting functionality included. One of the advantages is that it has several additional tools in the package (like rabin). The main advantage is that it can be install on ANY platform (mobile phones or PCs, Macs, Windows, Linux). You can statically examine, for example, an ARM binary on a PC for free, but you need to install radare on some ARM device to debug it.

Keywords: multiplatform, debugging, free, static analysis

IDA Pro 🍏, 🐧, 🌈

One of the best static analysis tools. However, it might be too much functionality for a beginner. It also has a scripting engine, so you can run your own Python scripts or scripts written on Ida’s proprietary language. ARM disassembling, hex rays (transforming the code to relatively human readable C++-like format) are available in the commercial version only.

Keywords: scripting, code translation, multiplatform, debugging

Hopper Disassembler 🍏

Works on Mac. Has a limited functionality for 30 minutes for Demo version. You can reopen the file after 30 minutes, but nothing is saved. Available for about 10k rubles. But it’s a life-time deal, which has all functionality (like hex rays analog - assembly translated into C++) included.

Keywords: scripting, code translation, multiplatform, debugging
