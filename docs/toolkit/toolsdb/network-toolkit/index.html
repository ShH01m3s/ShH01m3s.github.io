Impacket

Installing

pip install pcapy
pip install impacket

‚ùå Error: ip.get_th_sport doesn’t display ports

‚ùî Explanation: port number belongs to the transport layer. Impacket used to use it on IP layer, but now they’ve fixed it. Also, th in the function name means TCP, which is not always the case with transport layer (remember, UDP, ICMP).

‚úÖ Fix:

decoder = EthDecoder()
ether =  decoder.decode(data)
ip = ether.child()
transport = ip.child()
transport.get_th_sport()
# or
transport.get_uh_sport()
