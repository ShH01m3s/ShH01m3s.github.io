Several things make the mobile world a lot harder to deal with for an attacker and a lot more appealing.

Sandboxing

One of the biggest problems for an attacker is that each application is sandboxed. That means the apps are generally not allowed to talk to each other. While Android has a rather lax view on sandboxing, allowing several application mechanisms to gossip, iOS is much less permissive.
