Social Media And Messengers

File And Folder Activity Analysis

In order to detect and response to the incidents in a short time, there are playbooks which are basically guidelines. Some IR frameworks have these included in order to ease the process.

🏃‍♂️ Recent Activity Methodology

Address book 📖 Call history ☎️ (for mobile or Mac) SMS 💬 (for mobile or Mac) MMS 💬 🌄 (for mobile or Mac) Email ✉️ Web browser history Photos 📷 Videos 📹 Music 🎵 Documents 📃 Calendar 📆 Network communication 🌐 Maps 🗺️ Social networking 🦜 Deleted data 🗑️ Windows There are several artifacts indicating user activity.

💬❗️Notifications

Windows Toast notifications on Windows 10: C:\Users\%Username%\AppData\Local\Microsoft\Windows\Notifications contains wpndatabase.db and appdb.dat. Both can be opened with 🛠 SQLite Browser.

🗣 Voice Assistants

Windows Cortana is a great source of information. C:\Users\%Username%\AppData\Local\Packages\Microsoft.Windows.Cortana_xxxx\LocalState\ESEDatabase_CortanaCoreInstance\CortanaCoreDb.db contains user locations, reminders etc. Use a 🛠 SQLite browser to see the contents, or export to cvs and work in Excel.
