Cobalt Strike

IoC

  • Open TCP connection, port 50050
  • Get a default CS certificate

Components

For full information on how this works, see the article on Mandiant.

Team server

⛔️ Linux server only for the team server.

BEACON

⛔️ BEACON (agent) runs in memory, so, to detect it, a RAM dump is required.

Listener

Used by BEACON to connect to the Team server.

  • HTTP/HTTPS
  • DNS
    • Hybrid (DNS+HTTP)
    • Pure DNS
  • SMB
  • Raw TCP
  • Foreign (to talk to Meterpreter)
  • External (reverse TCP)
