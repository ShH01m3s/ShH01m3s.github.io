Search order hijacking

Following is the search order for modern systems:

  1. Dlls in the same direction as the executable that loads it
  2. KnowDLLs list HKLM\SYSTEM\CurrentControlSet\Control\WOW
  3. Application directory

  5. C:\Windows\system

Dll sideloading

Side-by side loading (SxS) mechanism to intriduce updated versions of DLL. Can be used to circumvent AV. Example: PlugX RAT. Legitimately signed exe uses SxS to load a malicious dll.

WMI Event Consumers

A little theory… . Event filter is a condition that should be met in order for something to happen. What’s the something? This something is an Event Consumer (script or executable). Binding is used to ties these two guys up. To setup this thing - use Powershell or mofcomp.exe (compiles MOF files into OBJECTS.DATA).

âš ï¸ WMI consumers run with SYSTEM privileges.

COM Hijack
