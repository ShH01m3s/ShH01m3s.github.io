Collection Checklist

  • Make sure the target media (evidence drive) is sterilised (ref. Identification/Assessment stage above).
  • Turn the machine off (better unplugging it to prevent data from being overwritten).
  • Attach the write blocker and the evidence drive.
  • Connect external HDD or SSD through the write blocker (if itâ€™s hardware-based)
  • Lauch some forensic distribution (for example, Paladin). Make an image of local HDD or use the original OS with all remote tools off (Bluetooth, AitDrop, WiFi, Cellular etc) and software-based write blocker on (if no hardware-blocker is used).
