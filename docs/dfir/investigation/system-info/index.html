AWS

On AWS there is a service that can be quieried for EC2 information. Official docs. Querying the EC2 IMDS for different instance information: curl -s "http://169.254.169.254/latest/meta-data/ami-id". This data can be quieried from within the instance.

List all buckets: aws s3 ls.

Turn off IMDS if you don’t need it for work.
