Mobile Device Characteristics

domain: general

doctype: article


ios

Mobile devices consist of the following parts:

  • Microprocessor
  • ROM (read-only memory)
  • RAM (random access memory) - execution of code happens here.
  • Radio module
  • digital signal processor
  • Microphone + speaker
  • Hardware keys and interfaces
  • LCD (Liquid cristal display)
  • NOR - OS can be stored here
  • NAND - OS can be stored here. Stacked NAND.
  • removable media (SDCards, microSDXC)
  • Non-cellular wireless communication (IrDA, Bluetooth, WiFi, NFC)

NFC is used, for example, in ApplePay.
