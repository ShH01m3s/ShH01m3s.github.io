🔪 Threat Hunting And Intelligence

⚠️ Incident Detection

Incident response starts with an event which could potentially be malicious. But first it needs to be proved. That’s when the incident detection comes into play. We analyse the event and determine whether it’s a malicious one making it an incident in case it’s confirmed. That’s where other IR stages start.

🥷 Incident Response

Once the incident is confirmed, you need to perfrom an investigation in order to be able fully contain it and remediate. This phase is also crucial to the threat intelligence collection which allows for more targeted hunting for adversaries accross the infrastructure.
