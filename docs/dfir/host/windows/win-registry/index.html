Windows Registry

domain: general

doctype: article


win

Prior to Windows NT, ini files were used. These were substituted with Windows registry to keep initialization and configurations. Some changes are also logged by the registry.

Structure

Hives are the first “folders” under the root folder. For example, under the root folder HKEY_LOCALMACHINE there is a set of folders. These are hives (purple rectangle). All the folders inside are keys and subkeys (green rectangle).

There are two types: system related files and user related. System related âš™ï¸:

User-related files mentioned above can be found at C:\Users\%Username%\NTUser.dat and C:\Users\%Username%\AppData\Local\Microsoft\Windows\UsrClass.dat. So, those mysterious XXX.dat files are actually a registry file.

SAM

This file cannot be edited through the RegEdit. Stores all the information about all the users: login information, password hashes, group information.

Software

Installed programs and applications (Microsoft\Windows\CurrentVersion\Uninstall), the type of OS (Microsift\WindowsNT\CurrentVersion), its install date, wireless networks, file association, login info, attached devices, autostart locations. Winlogon and LogonUI subkeys is about logon information.

There are two place for autoruns: Microsoft\Windows\CurrentVersion\Run and Microsoft\Windows\CurrentVersion\RunOnce.

System

Contains the following information:

  • Current control set
  • Computer name
  • Last shutdown time
  • Crash dump setting and location
  • Services set to run
