Mobile Acquisition Methods

domain: forensics

doctype: article


ios

In this article I’m studying different acquisition methods, tool available for each, pros and cons of each tool as well. Whenever I’ve tried some myself, I point it out explicitly.

Methods

Manual

Scrolling through the phone’s contents using GUI.

Tools 🛠:

Logical

Extracting logical storage objects: folders, files, directories. Sync with computer. Most often than not doesn’t include deleted items (other than cases when app has a bin). Doesn’t include unallocated space. Second best option.

Tools 🛠:

Physical

Bit by bit copy. Direct access to flash memory (non-volatile). Copies all data, including deleted and unallocated space. First best option.

Hex dumping. Requires uploading a modified boot loader into RAM on the device. The mobile devices is connected to the PC via flasher box. They were not designed for forensic purposes and one needs to know what she’s doing. Device is places in a diagnostic mode. However, flasher boxes have some limitations: requires rebooting (-> auth mechanisms activated and further analysis impossible), data encrypted, only certain ranges might be available, confisuing to use.

Tools 🛠:

JTAG. Some devices have special hardware intefaces to test hardware and firmware. One needs a JTAG adapter. One end connects to the mobile device’s motherboard and another one - to the PC. Here is a tool that I might buy in fiture in order to improve my skills in mobile forensics. Special software is used to interpret the results. JTAG adapter requests memory addresses from the mobile. Works for locked or damaged devices.

Tools 🛠:

Chip-Off. The memory is physically removed and a bit-by-bit copy is made. This is like imaging good old HDDs. But to provide the examiner with contiguous binary format, wear-leveling algorithm [4] must first be reverse engineered. There are lots of chip types and therefore, myrad of different raw data formats. JTAG is more preferable, since this technique (chip-off) mugh damage the device.

Tools 🛠:

Micro Read. For high profile methods only since requires an electronic microscope 🔬 and also a good technical knowledge. Using the microscope, one is observing the state of the gates of a NAND or NOR chips.

Tools 🛠:

No commercial tools currently exist to perform this task 😔.
