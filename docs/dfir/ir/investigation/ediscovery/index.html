eDiscovery

Forensics and eDiscovery are very close terms, but not the same. eDiscovery is collecting all information and diciding whether this information is relevant for the investigation.

When acquiring a mobile device, there are lots of information stored in different places. So, for example, say, we want to check the converstaion thread. But what if this conversation has started in iMessage and then moved to Outlook? Besides, there might be easier ways to acquire the same information (for example, we have an unlocked MacBook with the same iMessages).
