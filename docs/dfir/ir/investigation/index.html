🔎 Investigation

🗒️ Investigation Checklist

The Anatomy Of Investigation Identification/Assessment. Understand your scope (technical and legal aspects), your goals and people you are going to work with.

🗒 Data Collection Checklist

Logical Backup - doesn’t copy all, doesn’t copy slack, free space and deleted files. Imaging - bit-by-bit copy.

🗒 Data Analysis Checklist

First, have a template for an IR/forensics sheet. This can be done in Excel or other program alike.

📸 Imaging

Logical Backup - doesn’t copy all, doesn’t copy slack, free space and deleted files. Imaging - bit-by-bit copy.

eDiscovery

eDiscovery Forensics and eDiscovery are very close terms, but not the same. eDiscovery is collecting all information and diciding whether this information is relevant for the investigation.

Browser Info

Many browsers are based on Chromium engine, that’s why they will have similar artifacts: Chrome, Opera, new Edge, Brave, Vivaldi.

Host and Network Logs

Collection One of the most useful artifacts when properly configured and maintained, are log files. All devices keep some sort of logs, even mobile devices.

Network Traffic Analysis

Collection Most of the devices keep some sort of logs. As for the network related issues these are switches, routers, firewalls, IDS and IPS, web proxies, DC and authentication servers, DCHP server and application servers.

⚙️ Configuration Changes Analysis

Can be hard (no consistency). Are they different by accident or from malicous intent. Baseline reference (what the settings should be).

📬 Mail

Outlook The main artifacts are stored in C:\Users\%USERNAME%\AppData\Local\Microsoft\Outlook. Get OST and PST files. Contains messages, contacts, calendars, notes.
