Important Registry Keys

domain: general

doctype: list


windows

In this list I collect all the registry keys that I learned about or had experience with in the past and that are useful for forensics purposes (or may be not). I’ve had an idea to log all the keys since Windows regisrty is not fully documented still, but I don’t want to yet. May be after I retire. I will also add some interesting tips and manipulations with the registries or exploites. May contain errors.

Intro

Top-level hives

HKLM = HKEY_LOCAL_MACHINE

HKCR = HKEY_CLASSES_ROOT

HKCU = HKEY_CURRENT_USER

HKU = HKEY_USERS

HKCC = HKEY_CURRENT_CONFIG

Heap

HKCU\Software\Microsoft\Windows\CurrentVersion\Explorer\RunMRU\RecentDoc\ComputerDescription - systems this PC has recently talked to. Keywords: activity

USB_STORE - vendor name, type, unique id for all USBs ever used on PC.

HKLM\Security\Policy\Secrets - may contain RAS/VPN passwords, auto logon passwords and other sys passwords/keys. Also contains LSA secrets???
