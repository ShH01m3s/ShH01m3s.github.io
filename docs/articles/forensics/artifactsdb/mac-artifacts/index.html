Spotlight shortcuts

FSEvents

/.fseventsd # root of the system partition

Like UsnJrnl for Windows. Contains lots of logs gzip-compressed archive files. Log entries contain file ids which is useful for sequencing files and events. Since there are dozens of entries, it’s better to filter them. “Renamed” can mean the file was moved on the same volume.

Daily.out

/private/var/log/daily.out

Finder MRU

/Users/%username%/Library/Preferences/com.apple.finder.plist

AppStore Downloads

/Library/Receipts/InstallHistory.plist

Snapshots

tmutil to find and list all snapshots
