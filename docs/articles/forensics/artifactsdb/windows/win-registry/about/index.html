Windows Registry

domain: general

doctype: article


win

The registry is a dynamically expanding database of binary and ASCII data. Its primary purpose is to store configurations. One of the most important places to look for evidence. It’s a good place to look for configuration changes when performing cyber triage on the system.

Consists of hives. When a program is installed, reg entries are created. When this program is then deleted, its data not always deleted from the registry (although it’s supposed to).

Most of the tools sometime fail to show user created regisrty keys, so, it’s better to know the search string for that. Try some case specific keywords. To see the registry in a human viewable format, useregedit. Open Edit -> Find. Type the search word.

Here is my page in notion where I collect all the win reg keys I stumble upon in my research, study or work. I try to note its use and possible misuses as well.
