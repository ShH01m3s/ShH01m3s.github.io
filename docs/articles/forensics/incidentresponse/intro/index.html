Intro

CSIRT - Computer Security Incident Response Team

Data Breach is not the same as Security Breach. Data Breach cannot occur without a Security Breach. On the other hand, security breach happen without data breach. For example, ransomware is not a data breach since no data was leaked. However, it’s a security breach. So, we conclude that security breach is a more general term which includes a data breach.
