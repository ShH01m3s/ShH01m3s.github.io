Phases

This article is about the phases of mobile forensics.

Intake

The starting phase which included paper work on ownership and request forms. Everything boring and the most important from the court perspective goes here. Also, it’s critical not to damage the evidence. Answering the questions:

  1. What type of information the requestor is seeking?
  2. What is needed?
  3. What’s the scope

For example, I’m aquiring a mobile phone 📱 for analysis. I’m asked to find the evidence, that the person called number +12345679 (1,3). I take the phone but it’s locked. I ask to unlock it, because circumventing this limitation would take a lot of time and might corrupt the data (2).

Identification

Legal authority👨‍⚖️ First, one should keep in mind the legal aspects such as if the device is under warrant, then not all information can be analyzed 🚫.

Goals🎯 Then I’m supposed to determine the goals. Skip data irrelevant for the investigation.

Identification📱 Make, model and other id information, carrier, phone number. #todo: make a list of the required information.

Storages 💾. Removable and external data storage. If such storage exists (such as sdcard for Android), it should be removed and analysed separately with the traditional tools. But this card should be in the phone to ensure that the link between evidence is not broken.

❗ Don’t forget that mobile phones are physical devices which might contain physical evidence (for example, fingerprints).

Preparation

When the device was identified and the scope determined, make a list of tools, techniques and possible pitfols for that specific device.

Isolation

In case of mobile devices it’s very important, because mobile phones continuously collect different garbage 🗑️ like dogs which collect all sorts of crap 💩 while being walked. Some evidence might be overwritten. Also, keep in mind that many phones can be remotely wiped 🧹.

The first thing to do, is to put the phone in airplane ✈️ mode or a special radio frequency shielding cloth 👝 . Faraday bags block signals.

Processing

The phone should be aquired with a method that is repeatable and forensically sound. This phase starts only after the devices was isolated.

Physical acquisition is preferred, but when it’s not possible - logical acquisition. During logical acquisition (so that the evidences acquired will be admissable in court) it’s a good practice to have this process taped.

Verification phase

Handset data - RAM and ROM data from the phone. Extracted data sould be compared to handset data. Methods:

  1. Hashing algorithms. Every individually extracted file should be hashed. Every descrepancy should be explainable.
  2. Compare manually
  3. Compare with tools. Use multiple data to extract data and compare the results.
