Impacket

2020-09-29

Install

pip install pcapy
pip install impacket

Usage

❌ Error: ip.get_th_sport doesn’t display ports

❔ Explanation: port number belongs to the transport layer. Impacket used to use it on IP layer, but now they’ve fixed it. Also, th in the function name means TCP, which is not always a case with transport layer (remember, UDP, ICMP).

✅ Fix:

decoder = EthDecoder()
ether =  decoder.decode(data)
ip = ether.child()
transport = ip.child()
transport.get_th_sport()
# or
transport.get_uh_sport()
