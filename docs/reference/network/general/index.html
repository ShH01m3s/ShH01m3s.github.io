📚 General Network Terminology

domain: general

doctype: article


Network Topology

It’s a geometric arrangment of hosts and network devices. Can be physical (cables and PCs' layout) or logical (informaiton transmission methods).

Bus

To avoid latency, 5 PC max. Thin 10BASE2 (black) or thick 10BASE5 (yellow). 1 failes - all fail.

This is how it looks like on a diagram.

Hub

Options: UTP 10BASET (blue).

Low cost. 1 fails - net still on.

And this is a diagram so dear.

Ring

Options: Thicknet, fiber, optic.

Each node is a repater (token passing). 1 token - 1 network. 1 token is a series of bits containing control information.

Mesh

Options: Thicknet, fiber, optic.

1 fails - net still on.

Hybrid

Star-bus, star-ring

Architecture

Peer-to-peer.

Client-to-server.

Routing

Two types: static (no routing protocols needed) and dynamic (routing protocols needed). Routing protocols can be of distance vector, link state or hybrid. These also can be in autonomous system (IGP) or between autonomous systems (EGP).
