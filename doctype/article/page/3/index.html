Network Traffic Analysis

Collection Most of the devices keep some sort of logs. As for the network related issues these are switches, routers, firewalls, IDS and IPS, web proxies, DC and authentication servers, DCHP server and application servers.

Attacks 🗡

eDiscovery

eDiscovery Forensics and eDiscovery are very close terms, but not the same. eDiscovery is collecting all information and diciding whether this information is relevant for the investigation.

Firewalls 🔥

Firewalls monitor access from the outside network to the inside network. Traffic originating from inside the network would not pass through the firewall.???

IDS And IPS ✋

Host-based IDS look for local system configurations and behaviour, while network-based IDS look for network connections and looks for anomalies and common signatures.

Browser Info

Many browsers are based on Chromium engine, that’s why they will have similar artifacts: Chrome, Opera, new Edge, Brave, Vivaldi.

Host and Network Logs

One of the most useful artifacts when properly configured and maintained, are log files. All devices keep some sort of logs, even mobile devices.

Mail 📬

Outlook The main artifacts are stored in C:\Users\%USERNAME%\AppData\Local\Microsoft\Outlook. Get OST and PST files. Contains messages, contacts, calendars, notes.
