Imaging 🤏

Logical Backup - doesn’t copy all, doesn’t copy slack, free space and deleted files. Imaging - bit-by-bit copy.

FSEvents

Intro Directory events only 10.5-10.6. File events since 10.7. Each record is assigned a name which is constucted the following way: hex(last_event_id + 1).

Malware Analysis 🔎

Signed executable. It’s a rare case, but it exists. Around 3% of malware is signed. It’s due to the fact that some systems require executables to be signed.

Network Traffic Analysis

Collection Most of the devices keep some sort of logs. As for the network related issues these are switches, routers, firewalls, IDS and IPS, web proxies, DC and authentication servers, DCHP server and application servers.
